# EcoCivic Hub

## Problem
Lack of transparency and tracking in tree plantation initiatives.

## Solution
An AI-assisted platform to register, monitor, and reward tree plantation efforts.

## Tools Used
- Lovable AI (Frontend & Backend)
- Gamma AI (Presentation & Documentation)
- ChatGPT (Ideas & Prompts)

## Features
- Tree registration
- Photo-based monitoring
- Reward & recognition system
